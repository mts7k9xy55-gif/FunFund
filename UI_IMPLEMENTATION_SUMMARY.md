## 実行手順

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. Convex開発サーバー起動:
   ```bash
   npx convex dev
   ```

3. Next.js開発サーバー起動:
   ```bash
   npm run dev
   ```

4. ブラウザで確認:
   - プロジェクト一覧: `http://localhost:3000/public`
   - ダッシュボード: `http://localhost:3000/room` (Room選択後)

## 技術スタック

- **Next.js 16** (App Router)
- **React 19**
- **Tailwind CSS v4**
- **Geist Sans** (フォント)
- **Lucide React** (アイコン)

## 注意事項

- 一部のデータ（プロジェクトスコア、評価数など）はモックデータを使用しています
- 実際のデータ取得にはConvexクエリの実装が必要です
- プロジェクト詳細ページとプロジェクト作成フォームは今後の実装が必要です
